Acortador privacy-first

urlbreve

Acorta enlaces sin trackers, sin analytics invasivos y sin perfiles de visitantes. Código abierto, Django puro y API sencilla para herramientas web que necesitan enlaces cortos sin ruido.

  • Enlaces anónimos con ruta pública /a/<slug>/.
  • No guardamos IP, user-agent, referrer ni geolocalización.
  • Código auditable y licencia AGPLv3.

API-first para desarrolladores

Integra enlaces cortos en dashboards, bots, CRMs o herramientas internas.

Un endpoint JSON, respuesta clara y sin perfiles de visitantes. Puedes crear enlaces anónimos sin API key o usar X-API-Key para asociarlos a tu cuenta y activar rutas con namespace.
